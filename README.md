# 💰 Sistema de Banco em Python

Sistema simples de banco feito em Python no terminal.

## 🚀 Funcionalidades

- Cadastro de usuários
- Login
- Depósito de dinheiro
- Saque com verificação de saldo
- Visualização de saldo
- Sistema de logout

## 🧠 Tecnologias

- Python
- JSON

## ▶️ Como executar

```bash
python main.py
📁 Estrutura
main.py → sistema principal
usuarios.json → armazenamento de dados
